find largest files 

find / -type f -size +500M

Top 10 largest files/directories 

du -ah / | sort -rh | head -10 


du --> disk usage 
-a --> show all the files including directories
h --> human readable format 

du -ah / --> running on root file system 
sort -r --> reverse order i.e largest files first 

As running on root file system gives many permission denied messages 
sudo du -ah / 2>/dev/null | sort -rh | head -10 

or target specific paths for safer side 
du -ah /logs | sort -rh | head -10

Auto cleanup ,logs 

find /logs -name "*.log" -mtime +7 -exec rm -f {} \;

mtime = modificationtime -->checks when file was last modified 
mtime + 7 = older than 7 days
-exec - Execute command on each matched file 
syntax = -exec command {} \;
{} represents current matched file 
\; - marks end of execution

#!/bin/bash 
Url="https://example.com"
Status=$(curl -o /dev/null -s -w "%{http_code}" $Url)

if [ $Status -eq 200 ]
then 
	echo "Application is UP"
else 
	echo "Application is down"
fi


remember 
curl https://example.com downloads webpage content,
-o means output to file /dev/null --> sends to linux black hole file 
-s --> silent mods suppreses progess bar,download stats 
-w "%{http_code}" --> it tells curl , After requet completes ,pring specific info,now here print HTTP status code
